# Asset-Based Emergency Response Platform

A web application designed to support Root Causes in coordinating emergency food assistance, volunteer operations, and community resources. The platform provides role-based dashboards for staff and volunteers while allowing community members to discover and submit local resources.

## Features

- Google Authentication using Firebase Authentication
- Role-based access control (Staff, Caller, Driver, Packer)
- Participant management dashboard
- Driver assignment and delivery tracking
- Call logging for participant outreach
- Community resource directory
- Public resource submission with staff approval workflow
- Responsive web interface

## Technologies Used

- HTML
- CSS
- JavaScript
- Firebase Authentication
- Cloud Firestore
- GitHub Pages
